## Quick start guide

<Steps>
  <Step title="Invite Fire to your server">
    Add Fire to your Discord server using the invite link. Make sure to grant the necessary permissions for all features to work properly.
  </Step>
  
  <Step title="Set up moderation">
    Configure your moderation team using `/moderators add` and set up auto-moderation features to keep your server safe.
  </Step>
  
  <Step title="Explore features">
    Try out different commands to see what Fire can do. Use `/help` to see all available commands.
  </Step>
  
  <Step title="Customize for your server">
    Set up tags for frequently asked questions, create a ticket system for support, and configure ranks for your members.
  </Step>
</Steps>

## Popular features

### Moderation tools
Fire provides comprehensive moderation capabilities to keep your server safe and organized:
- Ban and kick members with optional reasons and durations
- Warn members and track warning history
- Mute members temporarily or permanently
- Purge messages with advanced filtering
- Lock down channels during emergencies

### Server utilities
Make server management easier with helpful utilities:
- View detailed server information and statistics
- Create beautiful custom embeds
- Set reminders for yourself or your team
- Check Minecraft player information
- Get user avatars and server icons

### Community engagement
Keep your members engaged and entertained:
- Play Tic-Tac-Toe with other members
- Ask the Magic 8-Ball for answers
- Use Google Assistant integration for queries
- Customize colors and explore emojis

### Custom content
Personalize your server experience:
- Create custom tags with rich embeds
- Set up self-assignable ranks (Premium)
- Configure reaction roles (Premium)
- Customize bot identity per server (Premium)
