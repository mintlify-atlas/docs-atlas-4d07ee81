## What is Fire?

Fire is a powerful Discord bot designed to be the only one you'll need for your server. From comprehensive moderation tools to fun commands and custom tags, Fire provides everything you need to manage and engage your community.

With support for slash commands, automatic moderation, detailed logging, and a robust ticket system, Fire helps you create a safer and more organized Discord server.

## Key features

<CardGroup cols={2}>
  <Card title="Moderation" icon="shield-halved">
    Keep your server safe with powerful moderation tools including bans, kicks, warnings, and automatic moderation for spam and unwanted content.
  </Card>
  <Card title="Tickets" icon="ticket">
    Create a professional support system with customizable ticket channels that help you organize member inquiries and support requests.
  </Card>
  <Card title="Custom tags" icon="tags">
    Build a knowledge base with custom tags that members can use to quickly access information, rules, or frequently asked questions.
  </Card>
  <Card title="Logging" icon="clock-rotate-left">
    Track everything happening in your server with comprehensive logging for messages, moderation actions, member updates, and more.
  </Card>
  <Card title="Utilities" icon="wrench">
    Access useful tools like avatar lookups, server info, embeds, role management, and Discord status checking.
  </Card>
  <Card title="Premium" icon="crown">
    Unlock advanced features including custom user identities, invite roles, vanity redirects, and server ranking.
  </Card>
</CardGroup>

## Why choose Fire?

<AccordionGroup>
  <Accordion title="All-in-one solution">
    Fire combines moderation, utilities, fun commands, and community features into a single bot, reducing the need for multiple bots and simplifying server management.
  </Accordion>
  <Accordion title="Slash commands ready">
    Fire fully supports Discord's modern slash command system, making commands easy to discover and use for all server members.
